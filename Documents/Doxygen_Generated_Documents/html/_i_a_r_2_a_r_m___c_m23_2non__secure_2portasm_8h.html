◆ vRaisePrivilege()

void vRaisePrivilege ( void )

Raises the privilege level by clearing the bit 0 of the CONTROL register.

Note
This is a privileged function and should only be called from the kenrel code.

Bit 0 of the CONTROL register defines the privilege level of Thread Mode. Bit[0] = 0 --> The processor is running privileged Bit[0] = 1 --> The processor is running unprivileged.
