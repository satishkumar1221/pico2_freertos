◆ RTOS_SUPPORT_INTERRUPT_STACK_GROWTH

#define RTOS_SUPPORT_INTERRUPT_STACK_GROWTH   ( 44 + 1 )

The number of words to extend the stack by when entering an ISR.

When entering an ISR we need to grow the stack by one more word than we actually need to save the thread context. This is because there are some functions, written in assembly cough memcpy() cough, that think it is OK to store words at SP[0]. Therefore the ISR must leave SP[0] alone even though it is normally not necessary to do so.
