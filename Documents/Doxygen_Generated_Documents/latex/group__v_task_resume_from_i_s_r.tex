INCLUDE\+\_\+x\+Task\+Resume\+From\+ISR must be defined as 1 for this function to be available. See the configuration section for more information.

An implementation of \doxylink{task_8h_a84a1584f29fb7736a1aa72ad5b3e9b44}{v\+Task\+Resume()} that can be called from within an ISR.

A task that has been suspended by one or more calls to v\+Task\+Suspend () will be made available for running again by a single call to x\+Task\+Resume\+From\+ISR ().

\doxylink{task_8h_aefbfd37c0661c3062fafd7334bff9aed}{x\+Task\+Resume\+From\+ISR()} should not be used to synchronise a task with an interrupt if there is a chance that the interrupt could arrive prior to the task being suspended -\/ as this can lead to interrupts being missed. Use of a semaphore as a synchronisation mechanism would avoid this eventuality.
