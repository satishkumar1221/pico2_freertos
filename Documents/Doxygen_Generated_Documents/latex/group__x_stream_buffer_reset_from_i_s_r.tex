An interrupt safe version of the API function that resets the stream buffer.

Resets a stream buffer to its initial, empty, state. Any data that was in the stream buffer is discarded. A stream buffer can only be reset if there are no tasks blocked waiting to either send to or receive from the stream buffer.

Use \doxylink{stream__buffer_8h_a5253429645c67902c2fc8682f8af0aac}{x\+Stream\+Buffer\+Reset()} to reset a stream buffer from a task. Use \doxylink{stream__buffer_8h_a6cd5f53639a304aba6a1d734cc2db0dd}{x\+Stream\+Buffer\+Reset\+From\+ISR()} to reset a stream buffer from an interrupt service routine (ISR).

config\+USE\+\_\+\+STREAM\+\_\+\+BUFFERS must be set to 1 in for Free\+RTOSConfig.\+h for \doxylink{stream__buffer_8h_a6cd5f53639a304aba6a1d734cc2db0dd}{x\+Stream\+Buffer\+Reset\+From\+ISR()} to be available.
