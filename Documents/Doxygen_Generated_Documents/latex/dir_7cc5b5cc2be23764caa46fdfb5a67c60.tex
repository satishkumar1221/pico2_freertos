The easiest way to use Free\+RTOS is to start with one of the pre-\/configured demo application projects. See \href{https://github.com/FreeRTOS/FreeRTOS/tree/main/FreeRTOS/Demo}{\texttt{Free\+RTOS/\+Free\+RTOS/\+Demo}} to find a list of pre-\/configured demos on multiple platforms which demonstrate the working of the Free\+RTOS-\/\+Kernel. \item The \href{./coverity}{\texttt{coverity}} directory contains a project to run \href{https://www.synopsys.com/software-integrity/static-analysis-tools-sast/coverity.html}{\texttt{Synopsys Coverity}} for checking MISRA compliance. This directory contains further readme files and links to documentation.
