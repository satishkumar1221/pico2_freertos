\#define RTOS\+\_\+\+SUPPORT\+\_\+\+INTERRUPT\+\_\+\+STACK\+\_\+\+GROWTH~( 44 + 1 )}

The number of words to extend the stack by when entering an ISR.

When entering an ISR we need to grow the stack by one more word than we actually need to save the thread context. This is because there are some functions, written in assembly {\itshape cough} memcpy() {\itshape cough}, that think it is OK to store words at SP\mbox{[}0\mbox{]}. Therefore the ISR must leave SP\mbox{[}0\mbox{]} alone even though it is normally not necessary to do so.
