\begin{DoxyItemize}

This port is generic and can be used as a starting point for other Armv8-\/A application processors. Therefore, the port {\ttfamily ARM\+\_\+\+CA53\+\_\+64\+\_\+\+BIT} is renamed as {\ttfamily ARM\+\_\+\+AARCH64}. The existing projects that use old port {\ttfamily ARM\+\_\+\+CA53\+\_\+64\+\_\+\+BIT}, should migrate to renamed port {\ttfamily ARM\+\_\+\+AARCH64}.

{\bfseries{NOTE}}

This port uses memory mapped interface to access Arm GIC registers.
